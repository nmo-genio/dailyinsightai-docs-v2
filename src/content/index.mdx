# 🧠 DailyInsightAI — Your Generative AI Journaling Assistant

**DailyInsightAI** is a personalized journaling and idea management app that leverages Generative AI to help you reflect better and grow smarter.

## ✨ What It Does

- 📝 **Summarizes** your journal entries using AI
- 📊 **Extracts insights** and highlights patterns
- ☁️ **Stores entries securely** in the cloud via MongoDB Atlas
- 🧭 **Guides daily reflection** with lightweight prompts
- 📈 **Tracks mood and ideas** over time

> ⚠️ The current version is CLI-based. A web interface is coming soon!

## 🛠️ Tech Stack

- **Language**: Python
- **AI**: OpenAI / LLMs
- **Database**: MongoDB Atlas
- **Packaging**: Poetry
- **Testing**: Pytest

## 🚀 Getting Started

Clone the repository and install dependencies:

```bash
git clone https://github.com/nmo-genio/dailyinsightai.git
cd dailyinsightai
poetry install
```

Run the CLI tool:

```bash
poetry run dailyinsightai
```
